Introduction To CSS

CSS(Cascading Style Sheets)is a styling language used to add beauty, creativity and personality to style and layout web pages

Ways to write CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS is the technique to define the single element with the insert style sheets in an HTML document.

Internal CSS is used to define a style for a single HTML page.
External CSS is a form of CSS which is used to add styling to multiple HTML pages at a time.
